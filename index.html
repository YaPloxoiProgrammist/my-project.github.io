Страница с персонализированными приветствиями

Этот сайт создан для демонстрации работы с всплывающими сообщениями. Нажмите на любую из кнопок ниже, чтобы отправить приветствие конкретному человеку. Особое сообщение подготовлено для Константина Сергеевича.

Как это работает

Каждая кнопка привязана к конкретному человеку. При нажатии срабатывает JavaScript-функция, которая определяет, какая кнопка была нажата, и показывает соответствующее сообщение.

Особенность

Для Константина Сергеевича реализовано специальное сообщение с просьбой поставить положительную оценку, в то время как для других отображается стандартное приветствие.

Технологии

Страница создана с использованием HTML, CSS и JavaScript. Дизайн адаптирован для мобильных устройств и включает плавные анимации.
